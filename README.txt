JSEcoin Captcha plugin adds the JSE captcha spam prevention tool to your website. The captcha plugin can be added to comments, login, registration and is compatible with contact form 7.

== Description ==

The JSE wordpress captcha plugin makes it as easy to prevent spam by adding the JSE captcha to your website. Instead of clicking on bridges users are prompted to complete a short mini-game.

By default it will work once activated on the comments section of your site. Additional settings are available for:

* Login page
* Registration page
* Contact Form 7

JSEcoin is a startup focusing on developing blockchain and web technologies. The JSE platform includes a number of tools for website owners to integrate cryptocurrency payments, advertising and content monetization. Further details are available at [JSEcoin](https://jsecoin.com "JSEcoin")

== Installation ==

In most cases the JSE captcha plugin should be easy to install via the admin panel directly.

Settings can be customised in the settings sub-menu.

Alternatively:
1. Upload `jsecoin-captcha.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the plugin settings by clicking JSEcoin on the main navbar

= How does the JSE captcha work? =

The JSE captcha collects user interaction patterns when the commentor plays the short game. This is then processed using Tensorflow machine learning algorithms and a risk score determines if the user passes the verification. Comment spam from automated queries and bots is prevented.
